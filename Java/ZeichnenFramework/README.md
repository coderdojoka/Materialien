# Framework
Das Framework entählt Klassen um ein fertiges Fenster mit einer Zeichenfläche zu erstellen. Ziel war es vom JDK zu abstrahieren, damit
man es als Anfänger einfacher hat.

# Bekannte Probleme
Beim Drücken von Knöpfen geht der Fokus für die Zeichenfläche verloren. Dadurch funktionieren Key-Events nicht mehr. Mit "Tab" lässt sich der Fokus wieder auf die Zeichenfläche setzen.
